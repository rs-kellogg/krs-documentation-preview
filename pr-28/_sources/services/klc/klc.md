# Kellogg Linux Cluster

The Kellogg Linux Cluster (KLC) is a set of high-memory Linux servers available to Kellogg researchers for interactive computing, large-scale data analysis, and batch jobs. KLC is part of Northwestern's Quest cluster, sharing its file systems, software module library, and SLURM job scheduler. Each node has 1.5–2 TB of RAM and up to 64 CPU cores.
