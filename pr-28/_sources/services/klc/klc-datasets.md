# Datasets on KLC

The following datasets are stored directly on the Kellogg Linux Cluster under `/kellogg/data/`. Access requires a KLC account. See [How to Access KLC](user-guide/klc-accessing) if you do not yet have one.

Contact [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu) to request access to a specific dataset.

| Dataset | Brief Description |
|---|---|
| [CME](../datasets/datasets/cme) | CME futures, options, and currency contracts (through 2007) |
| [Nielsen Marketing Data](../datasets/datasets/nielsen-marketing-data) | Consumer Panel, Retail Scanner, and Ad Intel |
