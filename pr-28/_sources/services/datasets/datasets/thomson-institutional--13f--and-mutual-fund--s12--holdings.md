## Description

The Institutional and Mutual Fund Holdings dataset contains quarterly observations of institutional and mutual fund common stock holdings from 1980 to present.

Mutual Funds data include information about common stock holdings and transactions of all mutual funds registered with the SEC plus 3,000 global funds. The data are quarterly from 1980 to present and are updated annually.

Some of these data overlap with information included in the Thomson (CDA/Spectrum) 13F Institutional common stock holdings. Thomson Financial includes the 13F tape identifier.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
