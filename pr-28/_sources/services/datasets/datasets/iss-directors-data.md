# ISS Directors Data

## At a Glance

Find information [here](https://wrds-www.wharton.upenn.edu/pages/get-data/institutional-shareholder-services-iss/directors/).

If you have questions, reach out to [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Description

ISS Directors Data is part of the ISS product suite (formerly published by IRRC, part of the RiskMetrics product by MSCI). ISS provides research on corporate governance, proxy voting, and corporate responsibility issues.

Universe of S&P 1500 companies —range of variables related to individual board directors (e.g., name, age, tenure, gender, committee memberships, primary employer and title, etc.)

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Time period** | 1996–present |
| **Geographic coverage** | US |
| **Unit of observation** | Director–firm–year |
| **Primary identifier** | CUSIP or ticker |
| **Commonly linked via** | CUSIP to CRSP/Compustat |
| **Key variables** | Director gender, age, ethnicity, compensation, committee membership, independence classification, interlocking directorships |
