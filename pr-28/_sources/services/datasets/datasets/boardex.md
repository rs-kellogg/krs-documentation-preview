# BoardEx

## At a Glance

Find information [here](https://wrds-www.wharton.upenn.edu/pages/about/data-vendors/boardex/).

If you have questions, reach out to [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Description

BoardEx contains valuable information on 1.7 million corporate executives and board members from over 2.2 million organizations. Biographical data includes details such as age, gender, positions held, educational qualifications, compensation and stock holdings, as well as insights into the executive's professional network.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Time period** | 1999–present |
| **Geographic coverage** | Global; over 20,000 companies worldwide |
| **Unit of observation** | Individual executive or director |
| **Primary identifier** | BoardID — company-level data; CompanyID — company-level data; DirectorID - individual-level data |
| **Commonly linked via** | ISIN, Ticker, CIK |
| **Key variables** | Demographics, education, career history, compensation, board and committee memberships, individual network connections |
