# Compustat Global

## At a Glance

Find information [here](https://wrds-www.wharton.upenn.edu/pages/get-data/compustat-capital-iq-standard-poors/compustat/).

If you have questions, reach out to [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Description

Compustat Fundamentals provides standardized North American and global financial statement and market data for over 80,000 active and inactive publicly traded companies that financial professionals have relied on for over 50 years.

Compustat Global is comprised of Non-U.S. and non-Canadian fundamental and market information on over 33,900 active and inactive publicly held companies.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Time period** | ~1987–present |
| **Geographic coverage** | 82 countries; >90% of world market capitalization |
| **Unit of observation** | Firm |
| **Primary identifier** | GVKEY — unique company ID |
| **Key variables** | Income statement, balance sheet, flow of funds, monthly prices, dividends, shares traded, currency translation rates |
