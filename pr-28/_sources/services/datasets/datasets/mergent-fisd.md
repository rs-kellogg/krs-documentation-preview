## Description

The Mergent Fixed Income Securities Database (FISD) for academia is a comprehensive database of publicly-offered U.S. bonds. FISD contains issue details on over 140,000 corporate, corporate MTN (medium term note), supranational, U.S. Agency, and U.S. Treasury debt securities and includes more than 550 data items. FISD provides details on debt issues and the issuers, as well as transactions by insurance companies. It is used to research market trends, deal structures, issuer capital structures and other areas of fixed income debt research.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/), under the [LSEG Mergent](https://wrds-www.wharton.upenn.edu/pages/get-data/lseg-mergent/) subscription. WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Time period** | 1995–2001 |
| **Geographic coverage** | US |
| **Unit of observation** | Security (debt issue) |
| **Primary identifier** | CUSIP |
| **Commonly linked via** | CUSIP, issue ID |
| **Key variables** | Issue characteristics, credit ratings, call/put/sinking fund schedules, coupon schedules, NAIC transaction pricing |
