# Dow Jones Averages & Total Return Indexes

## At a Glance

Find information [here](https://wrds-www.wharton.upenn.edu/pages/about/data-vendors/historical-spdji/).

If you have questions, reach out to [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Description

Closing prices and returns for the Dow Jones Industrial Average and other Dow Jones Indices.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/), under the [Historical SPDJI](https://wrds-www.wharton.upenn.edu/pages/get-data/dow-jones-averages-total-return-indexes/) subscription. WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
