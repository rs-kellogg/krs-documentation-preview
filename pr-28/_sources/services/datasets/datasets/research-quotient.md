## Description

RQ (Research Quotient) is a measure of a firm's R&D productivity, linking R&D spending to firm growth and market value. While RQ can be used to derive a firm's optimal R&D spending, its primary academic use is analyzing the effectiveness of firms' R&D. RQ offers universality, uniformity, and reliability advantages over patent-based measures for this purpose.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
