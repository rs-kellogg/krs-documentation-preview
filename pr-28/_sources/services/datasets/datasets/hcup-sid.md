## Description

The State Inpatient Databases (SID) are part of the family of databases and software tools developed for the Healthcare Cost and Utilization Project (HCUP). The SID includes inpatient discharge records from community hospitals in each state. The SID files encompass all patients, regardless of payer, providing a unique view of inpatient care in a defined market or state over time.

## Access
