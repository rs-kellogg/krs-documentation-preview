## Description

Health INFO PRO Data includes financial data filed by insurance companies, provided by the National Association of Insurance Commissioners (NAIC), covering 2001–2017.

## Access

This dataset is available on the [Kellogg Linux Cluster (KLC)](http://www.kellogg.northwestern.edu/rs/computing/kellogg-linux-cluster.aspx) under `/kellogg/data/naic_health_fin`.

To inquire about this resource, contact [Research Support](mailto:rs@kellogg.northwestern.edu).
