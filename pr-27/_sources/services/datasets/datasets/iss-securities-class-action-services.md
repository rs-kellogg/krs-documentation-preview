## Description

ISS Securities Class Action Services (ISS SCAS) offers a complete end-to-end litigation research and claims filing solution covering equities and fixed income securities across all markets.

## Access

These data are available on the Kellogg Linux Cluster under the `/kellogg/data/SCAS` directory. It contains two plain text files with a header row, delimited by the pipe (`|`) character. An Excel data dictionary is also available in that directory.

**Note:** The licensing agreement ran through April 2024. You may continue to work with the data beyond that date only to modify or reproduce papers or working papers that were published during the license period.
