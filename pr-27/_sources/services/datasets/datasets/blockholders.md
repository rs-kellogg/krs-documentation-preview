## Description

This dataset contains standardized data for blockholders of 1,913 companies. The data was cleaned from biases and mistakes typically observed in standard sources for this type of data. Blockholders' data is reported by firm for the period 1996–2001.

The data cleaning procedure is explained in detail by Jennifer Dlugosz, Rudiger Fahlenbrach, Paul A. Gompers, and Andrew Metrick in their study "Large Blocks of Stocks: Prevalence, Size, and Measurement."

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
