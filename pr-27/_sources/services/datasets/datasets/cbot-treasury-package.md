## Description

Updates to this data ended in 2007. Tick Data provides daily tick-by-tick data (time and sales) for all CBOT products. The following data is included: trade price, trade time, and volume traded (if available). End-of-Day Data provides quick access to the following daily prices for all CBOT products: open, high, low, close, settlement, volume, and open interest.

## Access

CBOT Treasury Package is available in the `/kellogg/data/cbot` directory on the Kellogg Linux Cluster (KLC). End-of-Day Data is in CSV format; Tick Data is stored as zipped CSV files.
