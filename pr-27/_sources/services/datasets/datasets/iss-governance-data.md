# ISS Governance Data

## At a Glance

If you have questions, reach out to [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Description

ISS Governance Data is part of the ISS product suite (formerly published by IRRC, part of the RiskMetrics product by MSCI). ISS provides research on corporate governance, proxy voting, and corporate responsibility issues.

ISS Governance & Takeover Defense provides data on classic takeover defense and other corporate governance provisions, including classified boards, cumulative voting, golden parachutes, poison pills, state takeover laws for S&P 1500 companies.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Geographic coverage** | US |
| **Unit of observation** | Firm–year |
| **Primary identifier** | CUSIP or ticker |
| **Commonly linked via** | CUSIP to CRSP/Compustat |
| **Key variables** | Corporate governance provisions, anti-takeover measures, G-index components |
