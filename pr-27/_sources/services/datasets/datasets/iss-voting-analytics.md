# ISS Voting Analytics

## At a Glance

## Description

ISS Voting Analytics is part of the ISS product suite and provides the industry's most complete solution for monitoring and analyzing voting policies, meeting results, and institutional voting patterns.

The database contains four main sub-datasets:

- **Vote Results for Russell 3000:** Data from 2003 onward for Russell 3000 companies; data from 2013 onward for other U.S. companies.
- **Global Vote Results:** Beginning 2013, meeting results for any company for which ISS procured the meeting agenda (except private equity, bondholder meetings, and a few others).
- **Institutional Vote Records:** Data from 2003 onward, covering 700+ institutions and 20,000+ funds.
- **N-PX Data:** Coverage from 2003; U.S. mutual fund voting records for all institutions filing the SEC Form N-PX.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.

## Data Coverage & Key Identifiers

| Attribute | Value |
|---|---|
| **Time period** | 2003–present (US); 2013–present (Global) |
| **Geographic coverage** | US and Global |
| **Unit of observation** | Meeting agenda item–institution–vote |
| **Primary identifier** | CUSIP or ticker |
| **Commonly linked via** | CUSIP to CRSP/Compustat |
| **Key variables** | Meeting date, agenda item, vote outcome, institutional voter, fund identifier |
