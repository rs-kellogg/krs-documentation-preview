## Description

As the primary regulator of the $3.7 trillion municipal securities market, the MSRB collects and makes transaction data publicly available through its Electronic Municipal Market Access (EMMA) system. The trades represent transactions by investors and dealers in the over-the-counter market for municipal securities issued by municipal entities, including states, counties, cities, and special tax districts.

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/). WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
