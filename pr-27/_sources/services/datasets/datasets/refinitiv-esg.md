## Description

LSEG ESG provides a comprehensive assessment of how companies perform on environmental, social, and governance (ESG) dimensions relative to their industry peers. The scoring framework evaluates performance across the three ESG pillars using over 600 quantitative and qualitative metrics, along with assessments of controversies within 10 thematic categories. As of August 2025, LSEG ESG covers approximately 90% of global market capitalization. The universe includes roughly 23,000 companies around the world, with final ESG scores available for about 15,000 companies.

In addition to core ESG scores, it provides Diversity & Inclusion (D&I) scores, which are calculated using a subset of ESG metrics drawn from the Workforce and Governance themes.

```{Important}
On WRDS, the entire Company Summary table is inactive and should not be used going forward. Certain variables in the other ESG tables have also been set to inactive. This data is incorrect and inactive items should not be used post 2014. They remain on WRDS for historical reference.
```

## Access

This dataset is available via [Wharton Research Data Services (WRDS)](https://wrds-www.wharton.upenn.edu/), under the [LSEG](https://wrds-www.wharton.upenn.edu/pages/get-data/lseg/refinitiv-esg/) subscription. WRDS offers both a web interface for small interactive queries and the WRDS Cloud for running batch programs. Faculty may request a temporary class account for course use.
