# Getting Help

The Research Support team at Kellogg enables and accelerates the academic research being pursued by our faculty and PhD students. We bring expertise from a wide variety of professional disciplines to help solve problems that can arise throughout the life cycle of a research project.

For all questions about KRS services, email [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).

## Kellogg Linux Cluster

To report an issue with KLC or get help troubleshooting, email [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu). 

For Quest-level issues (login nodes, scheduler, storage quotas), also see the [Northwestern RCDS Getting Help](https://rcdsdocs.it.northwestern.edu/start/getting-help.html) page.

## Datasets

For questions about dataset access, availability, or using a specific dataset, email [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu). Include the name of the dataset and a brief description of what you are trying to do.

For questions about datasets available through Northwestern Libraries (via SSDS), see the [Social Science Data Services](https://www.library.northwestern.edu/use-the-libraries/research-teaching/digital-scholarship/) page.

## Kellogg Data Cloud

For access requests or questions about KDC, email [rs@kellogg.northwestern.edu](mailto:rs@kellogg.northwestern.edu).
